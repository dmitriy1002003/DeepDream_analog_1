Аналог DeepDream на основе InceptionV3
О проекте

Проект посвящён исследованию методов генерации галлюциногенных изображений на основе сверточных нейронных сетей.

В качестве основы используется алгоритм DeepDream и предобученная модель InceptionV3.

Цель проекта — исследовать внутренние представления нейронной сети и визуализировать признаки, обнаруживаемые сетью на различных уровнях обработки изображения.

Состав репозитория
DeepDream_с_нуля.ipynb

Основная реализация алгоритма DeepDream.

Выполняет:

загрузку изображения;
создание модели DeepDream;
выбор слоев и фильтров;
генерацию итогового изображения.
Proba.ipynb

Экспериментальная среда.

Используется для:

подбора параметров;
анализа фильтров;
исследования поведения сети;
поиска интересных комбинаций признаков.
Используемые технологии
Python

InceptionV3

Принцип работы
Изображение подается на вход InceptionV3.
Выбираются внутренние слои сети.
Вычисляются активации выбранных фильтров.
Производится градиентный подъём по входному изображению.
Усиливаются обнаруженные признаки.
Формируется новое изображение с эффектом DeepDream.
Особенности проекта
не требует обучения модели;
использует готовые веса ImageNet;
позволяет исследовать внутренние признаки сети;
позволяет получать различные варианты галлюцинаций;
подходит для изучения механизмов работы сверточных нейронных сетей.
Итог

Проект демонстрирует механизм работы алгоритма DeepDream и позволяет исследовать влияние различных фильтров InceptionV3 на формирование визуальных галлюцинаций.
