# Аналог DeepDream на основе InceptionV3

# О проекте

Проект посвящён исследованию методов генерации галлюциногенных изображений на основе сверточных нейронных сетей.

В качестве основы используется алгоритм DeepDream и предобученная модель InceptionV3.

Цель — исследовать внутренние представления нейронной сети и визуализировать признаки.

# Используемые технологии
- Python
- TensorFlow
- InceptionV3
- NumPy
- Pillow
- Matplotlib
- Google Colab
# Принцип работы:
- Изображение подается на вход InceptionV3.
- Выбираются внутренние слои сети.
- Вычисляются активации выбранных фильтров.
- Производится градиентный подъём по входному изображению.
- Усиливаются обнаруженные признаки.
- Формируется новое изображение с эффектом DeepDream.
# Итог:
Проект демонстрирует механизм работы алгоритма DeepDream и позволяет исследовать влияние различных фильтров InceptionV3 на формирование визуальных галлюцинаций.
